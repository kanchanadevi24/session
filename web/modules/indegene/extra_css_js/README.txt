Extra CSS JS
--------------
This module provides functionality to add extra css and js to the site.


Installation
--------------
Download module
Upload module to sites/modules/contrib
Enable module


Permissions
------------
To make changes to the Extra CSS JS settings, users need the "administer
extra css js" permission.


Settings
----------
The configuration page can be found at admin/config/content/extra-css-js.
By Default The JS and CSS is added to all the themes, If you want it to appear
on specific theme/themes then select the themes from the list on configuration
page.


Limitations
-----------
This module adds css/js to all the pages rather than some selected pages.
